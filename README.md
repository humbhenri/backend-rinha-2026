# Rinha de Backend 2026

- https://github.com/zanfranceschi/rinha-de-backend-2026
- https://github.com/zanfranceschi/rinha-de-backend-2026/blob/main/docs/en/README.md

## Build

- `cd app`
- `go build`

## Run

Use `./rinha -s` to download the references file and preprocess the data into the filesystem.

After this, run it with `./rinha`. See `test.sh` on how to make a http request.
